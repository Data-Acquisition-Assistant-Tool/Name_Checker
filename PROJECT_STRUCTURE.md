# Project Structure

## Core Files
- `Namecheck.py` - Main application launcher
- `src/main.py` - Application entry point
- `src/file_utils.py` - File processing and batch renaming
- `src/excel_utils.py` - Excel file processing
- `src/ui/main_window.py` - Main user interface
- `src/ui/result_window.py` - Result display window
- `config/settings.py` - Application configuration

## Build Files
- `build_detailed.py` - Detailed build script
- `build_exe.bat` - Simple build batch file
- `快速打包Namechecker.bat` - Quick build script (Chinese)
- `Namechecker_1.2.spec` - PyInstaller configuration

## Archive
- `archive/NameCheck_legacy.py` - Original single-file version

## Dependencies
- `requirements.txt` - Python package dependencies

## Documentation
- `README.md` - Main documentation
- `PROJECT_STRUCTURE.md` - This file

## Test Directory
- `tests/` - Test files (currently empty)
